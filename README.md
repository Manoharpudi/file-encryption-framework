# 🔐 Python File Encryption Framework

## 📌 Description
This project is a secure file encryption system developed using Python. It allows users to encrypt and decrypt files using password-based cryptography.

## 🚀 Features
- Password-based encryption
- Secure file protection
- GUI using Tkinter
- Show/Hide password option
- Fast encryption & decryption

## 🛠 Technologies Used
- Python
- Cryptography Library
- Tkinter
- SHA-256 Hashing

## 📂 How it Works
1. Select a file
2. Enter password
3. Click Encrypt → file becomes unreadable
4. Click Decrypt → original file restored

## ⚠️ Important
- Same password must be used for decryption
- If password is lost, file cannot be recovered
